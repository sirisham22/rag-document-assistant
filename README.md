# RAG Document Assistant

An intelligent document retrieval system built with LangChain, FAISS, and OpenAI GPT models.
Designed to automate regulatory document discovery and customer support in banking.

## What It Does
- Ingests and chunks large documents
- Creates vector embeddings stored in FAISS
- Answers user queries using Retrieval-Augmented Generation (RAG)
- Serves responses via a FastAPI REST endpoint

## Tech Stack
Python | LangChain | FAISS | OpenAI | FastAPI | MLflow | Docker

## Results
- 96% precision in document retrieval
- 40% reduction in manual review effort
